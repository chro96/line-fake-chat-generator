# line-fake-chat-generator

架空のLINE風トーク画面をブラウザ上で作成し、PNG画像として保存できる静的Webアプリです。

## 使い方

`index.html` をブラウザで開くと動作します。ローカルサーバーで確認する場合は、プロジェクトルートで次を実行します。

```sh
python3 -m http.server 5173
```

その後、`http://localhost:5173` を開いてください。

## 主な機能

- 相手の名前、アイコン、背景画像、背景色、ステータスバー時刻の編集
- 相手/自分/日付/通知メッセージの追加、編集、並べ替え、複製、削除
- URLを含むメッセージのプレビューカード編集
- ブラウザ内の自動保存
- JSONエクスポート/インポート
- スマホ枠あり/なしのPNG保存
